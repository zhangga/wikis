## 主标题

AI 输出不该只是文档了：Claude 团队为什么开始偏爱 HTML？

## 推荐摘要

Claude Code 团队最近开始偏爱 HTML 输出。表面是格式之争，真正变化是 AI 交付物从线性文档变成可阅读、可点击、可导出的临时工作台。这篇给你一套判断框架和可直接照抄的提示词。

## 备选摘要

1. Markdown 没有死，但 AI 输出的下一步可能不再只是文档。Claude 这次引发的 HTML 讨论，真正值得看的不是格式优劣，而是人如何更好地复核 AI、参与 AI、把 AI 产物变成工作流。
2. 当 agent 一次能写出几百行计划和报告，真正的瓶颈就变成了“人还会不会认真读”。HTML 的价值不是漂亮，而是把 AI 输出变成更容易复核的界面。
3. 这篇不站队 HTML 或 Markdown，而是给一个实用判断框架：长期维护用 Markdown，快速理解和交互用 HTML，结构化事实最好保留在 JSON/YAML/CSV 里。
