## 正文

第一次装 OpenClaw，很多人以为跑完安装命令就结束了。

其实真正容易卡住的是后面两段：

1. `openclaw onboard --install-daemon` 里每一步到底怎么选
2. `openclaw config` 里模型和 API Key 怎么接

我把整个流程按 Windows 新手路线重新整理了一遍，做成了能直接照着操作的图文版：

- 安装命令怎么跑
- 初始化每一步怎么选
- 模型配置怎么接
- 最后怎么验证自己是真的装好了

这篇更适合这几种情况：

- 第一次装 OpenClaw
- 装到一半卡住
- Web UI 能打开，但模型没配好
