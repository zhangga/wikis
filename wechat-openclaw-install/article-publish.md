## 优化摘要

很多人第一次装 OpenClaw，以为把安装命令跑完就行，结果真正卡住的却是后面的初始化选项和模型配置。这篇按 Windows 新手视角，把安装、openclaw onboard --install-daemon、模型接入和重启验证完整顺了一遍，配上截图，适合直接照着做，尽量一次装好、不再反复重装。

## 备选摘要

1. 第一次装 OpenClaw，最容易让人放弃的不是安装脚本，而是后面那些“不知道该怎么选”的步骤。这篇把完整流程拆成安装、初始化、接入模型、重启验证 4 个动作，适合直接照着操作。
2. 如果你想把 OpenClaw 一次装好，这篇可以直接当操作稿：管理员 PowerShell 安装，openclaw onboard --install-daemon 走完初始化，openclaw config 接入模型，最后 openclaw gateway restart 验证生效。
3. 这不是一篇只给命令的安装帖，而是一篇把“哪里最容易卡、每一步该怎么选、装完后怎么确认可用”讲清楚的 OpenClaw 上手指南。
