# wikis

把日常随笔和学习内容整理为多平台发布稿件的工作仓库，覆盖**微信公众号**和**小红书**两个平台。

## 目录结构

```
wikis/
├── research/               # 选题研究、爆款拆解、写作 Playbook
├── wechat-<topic>/         # 微信公众号稿件（article.md / imgs/）
├── xhs-<topic>/            # 小红书稿件（note.md / caption.md / cards.md / imgs/）
└── AGENTS.md               # AI 协作规范（目录规则、内容标准、交付清单）
```

## 工作流

1. **选题 & 研究** — 在 `research/` 下完成平台数据研究和写作 Playbook 整理。
2. **起稿** — 按平台前缀新建目录，遵循 `AGENTS.md` 中的骨架和质量门槛。
3. **配图** — 所有图片放入对应主题 `imgs/` 目录，文件名加数字前缀保序。
4. **发布** — 微信需同时产出封面图和优化摘要；小红书需完成 `caption.md` 和 `cards.md`。

完整协作规范见 [AGENTS.md](AGENTS.md)。
